# 💜 MindaMate AI

MindaMate AI is an AI-powered emotional wellness companion built using the MERN Stack, Flask, and Ollama.

It helps users:
- Chat with an emotional AI companion
- Track moods and emotions
- Store emotional conversations
- View wellness insights and analytics
- Monitor emotional patterns over time

---

# ✨ Features

## 🤖 AI Emotional Chat
- Human-like AI wellness companion
- Emotion-aware responses
- Local AI powered using Ollama + Phi3 Mini

# ⚙️ Installation

## 1️⃣ Clone Repository

```bash
git clone https://github.com/Dimplekhatri/MindaMate-AI.git
```

---

## 2️⃣ Backend Setup

```bash
cd backend
npm install
npm start
```

Backend runs on:
```bash
http://localhost:9000
```

---

## 3️⃣ Frontend Setup

```bash
cd frontend
npm install
npm start
```

Frontend runs on:
```bash
http://localhost:3000
```

---

## 4️⃣ Flask AI Setup

```bash
cd flask_api
pip install -r requirements.txt
python predict_mood.py
```

Flask AI runs on:
```bash
http://localhost:5001
```

---

## 5️⃣ Run Ollama

```bash
ollama run phi3:mini
```

---

# 🔑 Environment Variables

Create `.env` file inside backend:

```env
DATABASE_URL=your_mongodb_url
JWT_SECRET=your_secret_key
```
